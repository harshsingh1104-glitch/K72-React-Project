## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/your-username/K72.git
```

Move into the project directory:

```bash
cd K72
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

---

## 🚀 Build for Production

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```
